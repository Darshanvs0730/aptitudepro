# Spring Boot Backend Help 

This folder contains the Java Spring Boot Backend. This engine acts as the middle-man between the React Website and our PostgreSQL database.

If you are a developer looking for how to work on this, here are the official links to read:
* [How to use Maven packages](https://maven.apache.org/guides/index.html)
* [Spring Web (How our APIs are processed)](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#web)
* [Spring Data JPA (How we talk to the database easily)](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#data.sql.jpa-and-spring-data)
* [Spring Security (How our passwords and JWT tokens work)](https://docs.spring.io/spring-boot/docs/current/reference/htmlsingle/#web.security)

## How to run it
Open your terminal inside this `Backend` folder and just type:
`mvn spring-boot:run`

It will fire up completely on its own on port `8080`.
