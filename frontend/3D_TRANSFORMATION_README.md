# 🚀 AptitudePro - Visual App Experience

## Simple Overview
We've completely upgraded the design of AptitudePro. Instead of just flat buttons and boring text, the main website features modern graphics, subtle animations, and 3D-like effects. The fundamental logic (PostgreSQL, Spring Boot, React Router) is the exact same, it just looks way cooler.

## ✨ The Key Design Features

- **Dark Mode by Default:** We chose a dark layout with glowing neon blue and purple accents to resemble modern IDEs and popular platforms like Discord. 
- **Glass-style Windows:** Instead of solid-color boxes, our dashboard boxes use a "Glassmorphism" effect, meaning they look slightly see-through over the background.
- **Animated Toast Messages:** Instead of using those annoying browser alerts (like "You are logged in!"), we custom-built animated notifications that slide in cleanly from the corner. 

## 🛠 Is performance affected?
No. The visual animations are optimized incredibly well. We chose generic lightweight Javascript tricks and smart CSS animations instead of downloading massive 3D-heavy graphic libraries, so the website will still load instantly even on slow computers or phones.

All the security features and test-taking logic we built originally works exactly the same!
