# AptitudePro Frontend

This folder contains the React.js website for the AptitudePro project.

## How to run the frontend
To run this part of the project on your computer:

1. Open this `frontend` folder in your terminal.
2. Type `npm start` and hit enter.
3. The website will open in your browser automatically at `http://localhost:3000`.

*Note: Make sure your Java Spring Boot backend is already running on port 8080!*

## Technologies Used
- React.js
- React Router DOM v7 (For navigating without refreshing)
- Axios (For talking to the Backend)
- Custom CSS (Dark Mode and Glassmorphism effects)
